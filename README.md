# Estruturas de dados

Repositório com exemplos das estruturas principais estrutudas de dados:

- `Vetores` (Arrays)
- `Listas`
    - `Listas encadeadas`  (Linked Lists)
    - `Listas duplamente encadeadas` (Doubly-Linked Lists)
- `Filas` (Queues)
- `Pilhas` (Stacks)
- `Conjuntos` (Sets)
- `Árvores` (Tree)
- `Tabelas espelhadas` (Hash Tables)
